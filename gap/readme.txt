"Geospace Analysis Package" project:
A collection of functions useful for data analysis in space physics. 

Installation
============
1) Copy the folder to your computer.
2) Open MatLab
3) Set path in MatLab to this folder and its subfolders. The paths can be
    set using either 'pathtool' command or using 'addpath()' function. See
    GAP_tutorial.pdf in gap_examples/tutorial folder for more information.
